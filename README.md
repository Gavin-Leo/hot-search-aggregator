# hot-search-aggregator

全网实时热搜聚合工具 —— 技能包。

## 功能

从多个主流平台（微博、抖音、B站、知乎、百度、今日头条）聚合实时热搜榜单，自动过滤娱乐圈/娱乐八卦/星座等无关内容，仅保留社会热点、时事新闻、科技、体育、电竞、数码等领域的热搜词条，并按统一格式输出前30条。

## 触发词

"查一下实时全网热搜""全网热搜榜""实时热点""热搜聚合""帮我看看现在网上都在关注什么""今日热搜""热点榜单""全网热议"

## 使用方式

在马维斯中说出触发词即可，例如：

> "帮我查一下实时全网热搜"
